Main Flow-of-Events
  1. The tank is controlled by arrows on the keyboard (top-bottom-right-left).
  2. Tank the tank moves  (top-bottom-right-left)
  3. For a shot, use the "space" on the keyboard
  4. A bullet is created and flies towards the direction
  5. The bullet hit the target
  6. Goal increment life, the bullet disappeared.
  7. If ( Goal life == 0 ) then
    1. Goal disappeared
    2. If ( Goal==Enemy ) then
      1. Enemy number increment
    3. If ( Enemy number == 0 ) then
      1. Victory
    4. If ( Goal == Gold && Goal == Player Tank ) then
      1. Game Over
  8. Else
    1. Goal changed the view
